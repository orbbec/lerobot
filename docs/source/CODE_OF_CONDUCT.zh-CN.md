# Contributor Covenant 行为准则（Code of Conduct）

## 我们的承诺

作为成员、贡献者与领导者，我们承诺让所有人参与我们的社区时都能获得无骚扰的体验，不论年龄、体型、可见或不可见的残障、族裔、性特征、性别认同与表达、经验水平、教育程度、社会经济状况、国籍、外表、种族、阶级、肤色、宗教，或性取向与性别取向。

我们承诺以促进开放、友善、多元、包容与健康社区的方式行事与互动。

## 我们的标准

有助于为社区营造积极环境的行为示例包括：

- 对他人展现同理心与善意
- 尊重不同的观点、立场与经验
- 提供并友善地接受建设性反馈
- 对自己的错误承担责任，并向因此受到影响的人道歉，并从经验中学习
- 关注的不只是个人最佳利益，也包括整体社区的最佳利益

不可接受行为的示例包括：

- 使用带有性暗示的语言或图像，以及任何形式的性关注或性示好
- 挑衅（trolling）、侮辱性或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息，例如住址或电子邮箱地址
- 在专业环境下可被合理认为不适当的其他行为

## 执行责任

社区领导者有责任澄清并执行我们的可接受行为标准，并将对其认为不适当、威胁性、冒犯性或有害的行为，采取适当且公正的纠正措施。

社区领导者有权也有责任移除、编辑或拒绝不符合本行为准则的评论、提交（commit）、代码、Wiki 编辑、issue 以及其他贡献，并将在适当情况下沟通其做出管理决定的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共场合以官方身份代表社区的情形。

代表我们社区的例子包括使用官方邮箱地址、通过官方社交媒体账号发布内容，或作为线上或线下活动中被指派的代表。

## 处理与执行

如发生辱骂性、骚扰性或其他不可接受的行为，可向负责执行的社区领导者举报：
feedback@huggingface.co。
所有投诉都将被及时且公正地审阅与调查。

所有社区领导者都有义务尊重举报者在任何事件中的隐私与安全。

## 执行指南

社区领导者将遵循以下“社区影响指南”（Community Impact Guidelines），以确定对任何其认为违反本行为准则的行为应采取的后果：

### 1. 纠正（Correction）

**社区影响（Community Impact）：** 使用不当语言，或其他被认为不专业或不受社区欢迎的行为。

**后果（Consequence）：** 社区领导者将私下发出书面警告，明确说明违规性质，并解释为何该行为不当。可能会要求公开道歉。

### 2. 警告（Warning）

**社区影响（Community Impact）：** 通过单次事件或一系列行为造成的违规。

**后果（Consequence）：** 发出警告，并说明继续违规将带来的后果。在指定时间内不得与相关人员互动，包括不得对执行本行为准则的人进行未经邀请的互动。这包括避免在社区空间内的互动，以及在社交媒体等外部渠道的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁（Temporary Ban）

**社区影响（Community Impact）：** 严重违反社区标准，包括持续的不当行为。

**后果（Consequence）：** 在指定时间内，临时禁止以任何形式与社区进行互动或公开沟通。在此期间，禁止与相关人员进行任何公开或私下互动，包括不得对执行本行为准则的人进行未经邀请的互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁（Permanent Ban）

**社区影响（Community Impact）：** 表现出持续违反社区标准的模式，包括持续的不当行为、对个人的骚扰，或对某些群体的攻击、贬损或煽动敌意。

**后果（Consequence）：** 永久禁止在社区内进行任何形式的公开互动。
